## Build

### Requirements

* gcc/++ or clang/++
* Make
* SYSTEMC_HOME environment variable that contains path to your systemc installation

### Building

``` bash
build.sh
```
will start compilation of the project

You can compile it manualy by running 
``` bash
make
```
and
``` bash
./cache
